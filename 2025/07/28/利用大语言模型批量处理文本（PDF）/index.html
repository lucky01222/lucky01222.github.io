利用大语言模型批量处理文本（PDF）

利用大语言模型批量整理历年考古发现数据：从PDF到结构化Excel

数据处理任务背景

撰写论文时常常面临大量非结构化数据整理的挑战。以收集"全国十大考古发现"数据为例，1990-2024年间的35份PDF文档包含了约350条珍贵的考古发现记录。手动整理这些信息不仅耗时耗力，还容易出错。本项目通过结合PDF解析技术和大语言模型(LLM)，实现了考古数据的自动化提取和结构化整理。

技术方案

整体架构

graph LR
    A[原始PDF文件] --> B[PDF文本提取]
    B --> C[LLM信息抽取]
    C --> D[JSON结构化]
    D --> E[Excel整合输出]

实现步骤

1.PDF文本提取

使用的提示词：

我现在遇到一个pdf处理问题，请你首先给我提供一份代码完成PDF的文字提取。然后我的文件路径在这个：1990-2024年历年十大考古发现/1990年全国十大考古发现.pdf ，我是使用jupyter，直接给我提供代码块即可

2.设计JSON数据结构

继续让大模型给出一个模拟的json格式数据，注意这一步尽可能把要求说明白些，以求得到格式准确的模拟数据

提示词：

好的，我已经完成提取，下一步我需要先设计一个所需要的json格式，我PDF的内容中1990年考古十大发现的内容，我需要整理成json格式数据，需要这些条目，文物名称，发现地点，不需要你帮我提取，请你帮我设计一份模拟的json格式数据，保证格式正确，有10个文物即可

4.批量处理流水线
