功能介绍

 本篇要介绍的是一个在github上开源的项目screenshot-to-code，该项目能够直接通过输入网站界面的截图将之转化为相似的界面，并提供相应的实现代码。

 项目具体细节可以在github上进行查看，这里给出该项目的链接：https://github.com/abi/screenshot-to-code
